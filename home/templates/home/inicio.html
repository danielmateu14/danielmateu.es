Sobre mí

Mi historia

Soy Daniel Mateu, desarrollador full stack especializado en Python y Django / Django REST Framework, centrado en construir software de gestión a medida que digitaliza y automatiza los procesos internos de las empresas.

Mi día a día va más allá de "montar una web": diseño y desarrollo plataformas que conectan sistemas que antes funcionaban aislados mediante integración de APIs de terceros (e-commerce, ERP y contabilidad, logística, telefonía, email y SMS, calendarios…), automatizo tareas repetitivas con RPA cuando no hay API disponible, y desarrollo funcionalidades con IA y agentes para apoyar la operativa diaria. Trabajo en un entorno internacional: cerca del 90% de los proyectos son para empresas francesas.

Mi stack habitual es Python, Django, Django REST Framework, PostgreSQL y JavaScript, con despliegue en producción (Hetzner, Railway, Render). Uso a diario herramientas de IA como Claude y Claude Code para acelerar el desarrollo, el modelado de datos y la depuración.

Idiomas: español (nativo), catalán (nativo), inglés (B1) y francés (A2, en formación).

Tecnologías principales:
Python Django Django REST PostgreSQL JavaScript RPA IA / Agentes Bootstrap CSS SQLite Git
